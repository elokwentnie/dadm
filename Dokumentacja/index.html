Cel i wprowadzenie do projektu

Projekt ten został zrealizowany w ramach przedmiotu “Dedykowane algorytmy diagnostyki medycznej”. Jego celem było stworzenie aplikacji analizującej sygnał EKG pozyskany z publicznych repozytoriów [1]. W ramach projektu zrealizowano osiem modułów, każdy odpowiedzialny za inną funkcjonalność: (1) ECG_BASELINE – filtracja sygnału, (2) R_PEAKS – detekcja załamków R, (3) WAVES – detekcja punktów charakterystycznych np. QRS-onset, QRS-end, (4) HRV1 - analiza czasowa i częstotliwościowa, (5) HRV2 – analiza geometryczna, (6) HEART_CLASS – kategoryzacja i klasyfikacja zespołów QRS, (7) ST_SEGMENT – analiza odcinka ST, (8) T_WAVE_ALT – oznaczanie załamków T, w których wykryto alternans. Aplikacja umożliwia wizualizację wyników wygenerowanych dzięki konkretnym modułom.

Wymagania systemowe oraz zastosowane technologie

Do realizacji projektu wybrano system operacyjny Linux (dystrybucja Ubuntu 20.04). Niniejszy projekt został napisany w języku C++ przy użyciu programu CLion 2020.2. Wykorzystano Cmake oraz kompilatory gcc i g++. Użyto następujących bibliotek:

  • Qt5
  • WFDB
  • C++ Standard Library
  • ALGLIB

Biblioteki wykorzystywane przez poszczególne moduły wylistowane są w dokumentacji tychże modułów.

Instalacja i uruchomienie aplikacji
Instalacja kompilatorów c i c++

sudo apt-get install gcc g++

Instalacja biblioteki WFDB

Należy pobrać i rozpakować źródło

curl https://physionet.org/physiotools/wfdb.tar.gz | tar xvz

Następnie przejść do rozpakowanego folderu i skonfigurować paczkę

cd wfdb-10.m.n

./configure

Następnym krokiem jest skompilowanie i instalacja paczki oraz przetestowanie paczki

make install

make check

Instalacja biblioteki Qt5

Instalacja paczek libgl1-mesa-dev oraz build-essential:

sudo apt-get install build-essential libgl1-mesa-dev

Pobranie pliku instalacyjnego ze strony http://download.qt.io/official_releases/online_installers/, a następnie przejście do katalogu, gdzie znajduje się pobrany plik i wpisanie komendy:

chmod a+x qt-unified-linux-x64-2.0.3-1-online.run

Następnie należy uruchomić program instalujący:

sudo ./qt-unified-linux-x64-2.0.3-1-online.run

I klikając “Next” dwukronie oraz “Install” zainstalować bibliotekę.

Instalacja biblioteki ALGLIB

Wpisanie w terminalu komendy:

sudo apt-get install libalglib-dev

Instalacja IDE

Pobranie pliku zip ze strony: https://www.jetbrains.com/clion/download/. Po wypakowaniu pliku przejście w terminalu do folderu bin:

~cd '/home/user/Programy/clion-2020.2.4/bin'

i uruchomienie pliku clion.sh za pomocą komendy w terminalu:

~./clion.sh
